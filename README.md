# 🎯 MysteryNumberGame

A browser-based number guessing game built with vanilla HTML, CSS, and JavaScript.


![Mystery Number Game](https://i.postimg.cc/TwTj4yLg/Screenshot-2026-04-07-212047.png)

---

## 📌 Overview

The player tries to guess a randomly generated secret number between **0 and 100**.
Each wrong guess costs a point and reveals whether the secret number is higher or lower. 
The game ends when the player guesses correctly or runs out of tries.

---

## 🎮 How to Play

1. Enter a number between **0 and 100** in the input field
2. Click **Check!** or press **Enter** to submit your guess
3. Follow the hints — 📉 Too Low / 📈 Too High / etc
4. Guess the correct number before your score hits zero
5. Click **Again!** to reset and play a new round

---

## ✨ Features

* 🎲 Random secret number generated on every new game
* 🕹️ Retro-style UI — interface inspired by classic old-school games
* 📊 Score tracking — starts at 20, decreases with each wrong guess
* 🥇 Highscore persists across rounds within the same session
* ✅ Input validation — rejects empty input, decimals, and out-of-range numbers
* ⌨️ Enter key support for faster gameplay
* 🎨 Visual feedback — background turns green on a win

## 🚀 Getting Started

No installation or build step required. You can run the game in two ways:

### Option 1 — Clone the repository

Clone the project using Git:

```bash
git clone https://github.com/saif-ahmedcs/MysteryNumberGame.git
cd MysteryNumberGame
```

Open the game in your browser:  
Double-click `index.html`

---

### Option 2 — Download the ZIP

1. Go to the repository page
2. Click **Code**
3. Select **Download ZIP**
4. Extract the ZIP file
5. Open `index.html` in your browser

---
Now you're ready to play!
